## 📦 Installation

### Quick Setup
```bash
# Clone the repository
git clone https://github.com/progmem-cc/miniblox.impact.client.updatedv2.git
cd miniblox.impact.client.updatedv2

# Install dependencies
npm install

# Build the client
npm run build
```

### Userscript Installation
1. Install a userscript manager (Tampermonkey, Greasemonkey, Violentmonkey)
2. Copy the contents of `tampermonkey.user.js`
3. Create a new userscript and paste the code
4. Save and enable the script
